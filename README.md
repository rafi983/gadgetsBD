# Gadgets BD - Static Next.js Templates

This is a component-based static Next.js site generated from your HTML templates.

## Routes

- `/`
- `/products`
- `/details`
- `/cart`
- `/bookings`
- `/create`
- `/forget-password`
- `/login`
- `/manage-list`
- `/payment-process`
- `/profile`
- `/register`
- `/review-modal`
- `/shops`
- `/success`

## Local Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quick Check

```bash
npm run lint
```

All pages are static placeholders with reusable components and mock data, ready for functionality wiring.
